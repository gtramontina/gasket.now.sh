gasket

shell pipeline as a service

Warning: This is a free service. Please be a good citizen and don't abuse it. Gasket runs the commands you specify on the same deployment box (contributions to make it more secure are very welcome), so please be careful not to expose any sensitive information. There will be, for sure, malicious people injecting something on this box to gather information, blow this up or both. This service is relying on Zeit's Now free tier and is deployed as a docker container, so it will be frozen from time to time and, hopefully, any screw ups would be solved (until someone f's it up again). Enjoy!
NOTE: if any command in the pipeline writes to sdterr, this service will consider it a failure and will result in a 400 with a dump of the contents of stderr.
